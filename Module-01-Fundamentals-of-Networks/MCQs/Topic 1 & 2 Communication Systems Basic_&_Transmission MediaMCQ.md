**Topic 1: Communication Systems Basics (30 MCQs)**



1\. What is the primary purpose of a communication system?



A. Data storage

B. Data processing

C. Data transmission

D. Data encryption



✅ Answer: C

Explanation: Communication systems are designed to transmit information from sender to receiver efficiently.



2\. Which component converts data into signals?



A. Receiver

B. Medium

C. Transmitter

D. Noise



✅ Answer: C

Explanation: The transmitter encodes data into signals suitable for transmission.



3\. What is noise in communication systems?



A. Signal strength

B. Interference

C. Data packet

D. Protocol



✅ Answer: B

Explanation: Noise refers to unwanted disturbances that affect signal quality.



4\. Bandwidth refers to:



A. Signal delay

B. Data capacity

C. Packet size

D. Error rate



✅ Answer: B

Explanation: Bandwidth defines how much data can be transmitted per unit time.



5\. Which signal type is continuous?



A. Digital

B. Analog

C. Binary

D. Packet



✅ Answer: B

Explanation: Analog signals vary continuously over time.



6\. Which is NOT a communication system element?



A. Sender

B. Receiver

C. Protocol

D. Compiler



✅ Answer: D

Explanation: Compiler is unrelated to communication systems.



7\. Full-duplex communication means:



A. One-way transmission

B. Two-way but not simultaneous

C. Two-way simultaneous

D. No transmission



✅ Answer: C

Explanation: Both sender and receiver transmit simultaneously.



8\. Half-duplex communication example:



A. Telephone

B. Walkie-talkie

C. Email

D. Fiber link



✅ Answer: B

Explanation: Only one side transmits at a time.



9\. Simplex communication is:



A. Two-way

B. One-way

C. Full-duplex

D. Packet-based



✅ Answer: B

Explanation: Data flows in only one direction.



10\. Latency refers to:



A. Bandwidth

B. Delay

C. Noise

D. Encoding



✅ Answer: B

Explanation: Latency is the time delay in data transmission.



11\. Throughput is:



A. Maximum speed

B. Actual data rate

C. Signal strength

D. Packet loss



✅ Answer: B

Explanation: Throughput is real achieved data transfer rate.



12\. Which affects signal degradation most?



A. Distance

B. Protocol

C. Port number

D. MAC address



✅ Answer: A

Explanation: Longer distance increases attenuation.



13\. Modulation is:



A. Error correction

B. Signal conversion

C. Routing

D. Switching



✅ Answer: B

Explanation: Converts data into signals for transmission.



14\. Demodulation occurs at:



A. Sender

B. Receiver

C. Router

D. Switch



✅ Answer: B

Explanation: Receiver converts signals back to data.



15\. Bit rate is measured in:



A. Hz

B. bps

C. dB

D. Volts



✅ Answer: B

Explanation: Bits per second measures data rate.



16\. Which is an example of digital signal?



A. Sine wave

B. Binary data

C. Analog voltage

D. Radio wave



✅ Answer: B

Explanation: Digital signals use discrete binary values.



17\. Signal attenuation means:



A. Amplification

B. Strength loss

C. Noise removal

D. Encryption



✅ Answer: B

Explanation: Signal weakens over distance.



18\. Which improves signal quality?



A. Noise

B. Amplifier

C. Delay

D. Packet loss



✅ Answer: B

Explanation: Amplifiers boost signal strength.



19\. What is jitter?



A. Delay variation

B. Packet loss

C. Bandwidth

D. Noise



✅ Answer: A

Explanation: Jitter is variation in packet delay.



20\. Which device regenerates signals?



A. Router

B. Switch

C. Repeater

D. Hub



✅ Answer: C

Explanation: Repeaters regenerate weakened signals.



21\. Error detection ensures:



A. Speed

B. Accuracy

C. Routing

D. Encryption



✅ Answer: B

Explanation: Detects corrupted data.



22\. Which layer handles encoding?



A. Application

B. Presentation

C. Physical

D. Network



✅ Answer: C

Explanation: Physical layer deals with bit transmission.



23\. Data integrity means:



A. Fast transmission

B. Accurate data

C. Large packets

D. Encryption



✅ Answer: B

Explanation: Data remains unchanged during transmission.



24\. Signal-to-noise ratio measures:



A. Speed

B. Quality

C. Delay

D. Size



✅ Answer: B

Explanation: Higher SNR means better signal quality.



25\. Which medium is fastest?



A. Copper

B. Fiber

C. Wireless

D. Coaxial



✅ Answer: B

Explanation: Fiber offers highest speed and lowest loss.



26\. Packet switching is used in:



A. Telephone

B. Internet

C. Radio

D. TV



✅ Answer: B

Explanation: Internet uses packet-switched communication.



27\. Circuit switching example:



A. Email

B. Internet

C. Telephone

D. FTP



✅ Answer: C

Explanation: Dedicated path is established.



28\. Which is a communication protocol?



A. HTTP

B. CPU

C. RAM

D. GPU



✅ Answer: A

Explanation: Protocol defines communication rules.



29\. Encoding converts:



A. Signal to data

B. Data to signal

C. Packet to frame

D. Frame to packet



✅ Answer: B

Explanation: Encoding prepares data for transmission.



30\. Which factor affects throughput most?



A. Delay

B. Bandwidth

C. Noise

D. All of the above



✅ Answer: D

Explanation: Throughput depends on multiple factors.





**Topic 2: Transmission Media (Wired \& Wireless) – 30 MCQs**



1\. Which transmission media uses light for data transmission?



A. Twisted Pair

B. Coaxial

C. Fiber Optic

D. Radio



✅ Answer: C

Explanation: Fiber optic cables transmit data using light signals, offering very high speed and low attenuation.



2\. Which cable type is most susceptible to EMI (Electromagnetic Interference)?



A. Fiber

B. UTP

C. STP

D. Coaxial



✅ Answer: B

Explanation: UTP lacks shielding, making it more vulnerable to EMI.



3\. Which cable provides shielding against interference?



A. UTP

B. STP

C. Fiber

D. Twinax



✅ Answer: B

Explanation: STP includes shielding to reduce EMI.



4\. What is the maximum length of UTP cable (Ethernet standard)?



A. 50m

B. 100m

C. 500m

D. 1km



✅ Answer: B

Explanation: Ethernet over UTP typically supports up to 100 meters.



5\. Which connector is used with UTP cables?



A. BNC

B. RJ45

C. SC

D. LC



✅ Answer: B

Explanation: RJ45 connectors are standard for Ethernet cables.



6\. Which fiber type supports longer distances?



A. Multimode

B. Single-mode

C. Both equal

D. None



✅ Answer: B

Explanation: Single-mode fiber supports long-distance transmission due to less dispersion.



7\. Which media is immune to EMI?



A. Copper

D. UTP
